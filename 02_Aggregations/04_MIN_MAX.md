# MIN() and MAX()

## Definition

MIN() returns the smallest value.

MAX() returns the largest value.

---

## Syntax

```sql
SELECT MIN(column_name)
FROM table_name;
```

```sql
SELECT MAX(column_name)
FROM table_name;
```

---

## Business Use Cases

- Lowest salary
- Highest salary
- Earliest order
- Latest order

---

## Interview Tip

MIN() and MAX() work with numbers, dates, and text values.
